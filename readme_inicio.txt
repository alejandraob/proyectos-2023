Recuerda que para usar git tus comandos a usar es:
git init (crea repositorio y archivo oculto de git en tu equipo) //comprobar despues poniendo el nombre de archivo
git config --global user.mail "tu mail@" (debes ingresar tu mail) //Habilitamos la credenciales de usuario
git config --global user.name "tu usuario" (debe ingresar usuario) // Habilitamos segunda credencial de usuario
git add . (agrgeamos todo el contenido de la carpeta) //comprobar si es antes de commit o despues...creo que antes
git commit -m "mensaje del comentario" (creamos un comentario para la subida)
git push -u origin main (subimos al repositorio que se encuentra en github)
